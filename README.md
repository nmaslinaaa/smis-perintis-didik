
A web-based Student Management Information System (SMIS) developed for Pusat Tuisyen Perintis Didik to digitalize and improve the management of student information, attendance tracking, and academic performance monitoring.

This system replaces manual processes such as paper-based attendance records and spreadsheet data management with a centralized digital platform.

The system supports three main user roles:

1️⃣ Administrator

- Manage student and teacher information

- Verify student registration

- Monitor attendance

- Manage fee status

- Generate reports

2️⃣ Teachers

- Record student attendance

- Evaluate student performance

- Track syllabus coverage

3️⃣ Parents

- Register their children

- View student attendance

- Monitor academic progress

- Check fee status
